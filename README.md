# Video-Downloader
输入链接，获取各视频站视频所在的真实地址：
可设定videoType ： n（标清）、h（高清）、s（超清）

目前暂时支持：优酷，土豆，乐视，搜狐（后续。。。。看心情添加）

下载地址：http://pan.baidu.com/s/1dEi79lR

PS：
就是丑！！！界面巨丑，不能再丑。不过就是单纯的娱乐一下，有兴趣就更新下。。。。
